# 開發環境

只列出函數庫版本.

  - Python 3.6.5 
  - OpenCV 3.4.1
  - Numpy 1.14.2

安裝OpenCV套件 [參考](https://www.scivision.co/install-opencv-python-windows/)

```sh
$ python -m pip install opencv-python
$ python -m pip install opencv-contrib-python #Extra contributed modules
```

安裝Numpy套件

```sh
$ python -m pip install numpy
```
